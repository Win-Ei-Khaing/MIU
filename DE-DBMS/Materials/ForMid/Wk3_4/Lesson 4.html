Lesson 4 - SQL - Data Definition

Purpose: -  Read and write SQL queries, translate questions in English into SQL.

Base Data types in SQL

Boolean, character, bit (removed from SQL:2003), exact numeric, approximate numeric, datetime, interval, large object.

SQL Commands for Data Definition (defining database structure)

CREATE - to create a database schema or table;
ALTER - to change the structure of a table once it has been created;
DROP - to delete a table

CREATE TABLE <TABLE NAME>
(<COLUMN NAME> <DATA TYPE> [(<SIZE>)] <COLUMN CONSTRAINT LIST> [,...]) 
NULL or NOT NULL
DEFAULT
UNIQUE
PRIMARY KEY | PRIMARY KEY (column [,...])
FOREIGN KEY REFERENCES <TABLE NAME> (<COLUMN NAME> [,...])
CHECK

ALTER TABLE <TABLE NAME> ADD|DROP|MODIFY (COLUMN SPECIFICATION LIST) 

DROP TABLE <TABLE NAME>

 
CREATE VIEW <TABLE NAME>
AS <QUERY>

 DROP VIEW <VIEW NAME>

Integrity Enhancement Feature

Required data NOT NULL of CREATE/ALTER TABLE.
Domain constraint CHECK clause of CREATE/ALTER TABLE and CREATE DOMAIN.
Entity integrity PRIMARY KEY (and UNIQUE) clause of CREATE/ALTER TABLE.
Referential integrity FOREIGN KEY clause of CREATE/ALTER TABLE
General constraints CHECK and UNIQUE clauses of CREATE/ALTER TABLE and (CREATE) ASSERTION
